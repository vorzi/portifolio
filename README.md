# My portifolio

# REQUIREMENTS
Node.js (or another javascript runtime that works on typescript and nextjs)  
MySQL 8+ (Or MariaDB)

# How to install?
```bash
git clone http OR (Download Zip)
cd portifolio
bun install
bun dev
```

change the `.env.example` (and rename it `.env.example` to `.env`)

and put the `table.sql` into mysql

```bash
mysql <your login credits> < table.sql
```
